--- layout: project title: "Creolet" description: "Simple Diary Entry Messenger Chat-Bot" tech: - python - API - flask ---

Creolet is a chat-bot deployed on Facebook Messenger to record diary entries in form of text and other multimedia, in a database hosted on Heroku. The database server uses flask to serve database queries.

There are multiple components involved-

  • Chat-bot Deployment on Facebook
  • Webhook written in python
  • Microsoft LUIS API for NLP
  • A Database
  • A Database Server
  • A Hosting Platform
